# Energy Equity Dashboard Starter

This is a starter template for a hackathon project focused on **Community Energy, Equity and Sustainability**.

## Stack

- Next.js
- Tailwind CSS
- MongoDB Atlas
- Clerk Auth
- Recharts

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create a `.env.local` file:

```bash
MONGODB_URI=your_mongodb_atlas_connection_string
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key
CLERK_SECRET_KEY=your_clerk_secret_key
```

3. Run the project:

```bash
npm run dev
```

4. Open:

```bash
http://localhost:3000
```

## Product idea

This dashboard helps utilities, municipalities, and community partners identify neighbourhoods with high energy burden, outage risk, heat vulnerability, and sustainability opportunity.

The goal is to help organizations like Alectra or Estri decide where energy support, infrastructure investment, and community outreach should go first.
